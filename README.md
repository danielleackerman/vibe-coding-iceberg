## üßä Iceberg Layers

Each layer of the iceberg represents a part of the tech stack:

1. **UI & Interaction** ‚Äì React, Tailwind, Subframe
2. **App Logic** ‚Äì Form handling, routing, validation
3. **Services & APIs** ‚Äì Firebase, Supabase, Stripe, OpenAI
4. **Data Modeling** ‚Äì Prisma, Drizzle, SQL
5. **Infrastructure & CI/CD** ‚Äì Vercel, Netlify, Astro, serverless
6. **AI Prompting** ‚Äì How language connects all the layers



## üí° Why Am I Sweating?

"Vibe coding" is the emerging practice of using tools and prompts to generate real software ‚Äî not just from vague ideas, but from **clear specifications as prompts**.

This repo exists to:

- Track those tools
- Understand how they work under the hood
- Collect prompt examples that reveal the hidden layers
- Explore the full stack of modern development as an iceberg
